# 🚀 Invoice Pilot

A full-stack invoice management system built with:

- ⚡ Next.js (Frontend)
- 🟢 Node.js + Express (Backend)
- 🔥 Firebase (planned)
- 💾 In-memory storage (current)

---

## ✨ Features

- 📊 Dashboard with invoice tracking
- ➕ Create invoices
- 💰 Auto amount calculation (based on duration)
- ⚠️ Due invoice detection
- ✅ Mark invoices as paid
- 🔍 Filter (All / Due / Paid)

---

## 🖥️ Tech Stack

- Frontend: Next.js (App Router)
- Backend: Express.js
- Styling: Inline CSS (can upgrade to Tailwind)
